# Portail PS – Frontend

##  Description
Frontend de l’application Portail PS développée avec React et Redux.
Il permet aux prestataires de soins de consulter les droits des assurés via une interface moderne et sécurisée.

## Technologies
- React JS
- Redux
- Tailwind  CSS
- Axios

## Fonctionnalités
- Authentification
- Consultation des droits
- Interface responsive
- Connexion API sécurisée
